# WebWrapper

Android app that lets you save websites and open them in an in-app `WebView`.

## Features

- **Website launcher**: Add and open saved URLs.
- **History list**: Edit entries (custom name, tag, optional logo).
- **Tag filtering + search**: Filter saved sites by tag and search text.
- **Pattern lock**: Optional app lock (verify on launch when enabled).
- **Home screen widget**: AppWidget provider included.

## Tech stack

- **Language**: Kotlin
- **Min/Target SDK**: 24 / 34
- **UI**: ViewBinding + Material Components
- **WebView**: `androidx.webkit:webkit`
- **Native code**: CMake + NDK (C++17)

## Requirements

- **Android Studio**: Iguana or newer recommended
- **JDK**: 17
- **Android SDK**: 34
- **NDK/CMake**: Required (project uses `externalNativeBuild`)

## Build & run

- **Android Studio**:
  1. Open the project folder.
  2. Let Gradle sync.
  3. Select the `app` configuration.
  4. Run on an emulator or device.

- **Command line**:
  ```bash
  ./gradlew :app:assembleDebug
  ```

## Permissions

Declared in `app/src/main/AndroidManifest.xml`:

- `INTERNET`
- `CAMERA` (optional)
- `RECORD_AUDIO` (optional)

## Project structure

- `app/` Android application module
- `app/src/main/java/com/nan/webwrapper/` app source
- `app/src/main/cpp/` native code (CMake)

## Notes

- `local.properties` is expected to be generated by Android Studio (SDK path). Do not commit secrets.
